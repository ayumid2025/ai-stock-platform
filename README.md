# AI-Assisted Stock Trading Platform

A web application that allows users to trade stocks with AI-powered insights.

## Features (Planned)
- User authentication
- Real-time stock data and charts
- AI price predictions
- Paper trading via Alpaca API
- Portfolio tracking

## Tech Stack
- Backend: Python, Flask, SQLAlchemy
- Frontend: HTML, CSS, JavaScript, Chart.js
- Database: PostgreSQL (prod), SQLite (dev)
- APIs: Alpaca Markets
- AI: scikit-learn
